# Description
> "i" 키를 입력해서 인벤토리에 진입할 수 있고, 오른쪽 상단의 Create Item을 통해 아이템을 생성할 수 있음. <br/>
> 생성된 아이템은 부위가 나뉘어지고, 장비는 각 부위에만 장착할 수 있음. <br/>
> 장착할 때 능력치가 더 낮거나 높다는 알림이 나타남. <br/>
